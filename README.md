# Performance Calculator

A web-based KPI Performance Calculator built with HTML, CSS, and JavaScript.

##  Features
- Add, edit, and remove employees
- Automatically calculate weighted KPI scores
- Save data locally using localStorage
- Export results to CSV
- Supports monthly and yearly performance tracking
- Responsive design for all devices



## How to Use
1. Open `index.html` in a browser or visit your GitHub Pages link.
2. Add employees and enter their KPI data.
3. Click **Export Results** to download performance reports as CSV.
